# Smart Student Performance Analyzer
## Overview
Smart Student Performance Analyzer is a project based on Machine Learning. It is designed to predict the end marks of a student and to state whether a student will pass the course or fail.  It takes the following as input a student marks, number of study hours of a student and number of sessions attended.
